GATE Http 2.0 Firmware

Pobierz oprogramowanie beta i sprawdź co nowego

Wersja Beta

v1.4.2 (build 2346)

20 listopada 2023

Firmware dostępny do aktualizacji za pomocą paczki zip.

Proces aktualizacji i uruchomienie zgodne z User Manual - Rozdział VI, Punkt 10.2.B.

Opis funkcjonalności: Moduł GATE HTTP - User Manual.

Co nowego

  • Nowy obiekt wirtualny HEOS:
    • Integracja z urządzeniami Denon HEOS.
  • Nowy obiekt wirtualny DenonMarantzAVR:
    • Integracja z urządzeniami Denon oraz Marantz AVR.
  • Wsparcie dla logów diagnostycznych poprzez telnet. Nowa metoda (SetTelnetLogLevel) oraz cecha (TelnetLogLevel) do ustawiania poziomu logowania.
  • Zmiany dla obiektu HttpRequest:
    • Nowa cecha ResponseHeaders - dostęp do nagłówków odpowiedzi HTTP.
    • Nowa cecha: IsActive - ukazuje, czy aktualnie aktywne jest zapytanie HTTP.
  • Zmiany dla obiektu HttpListener:
    • Nowa cecha ResponseHeaders oraz metoda SetResponseHeaders - możliwość ustawienia nagłówków odpowiedzi HTTP.
    • Nowa cecha RequestHeaders - dostęp do nagłówków zapytań HTTP.
  • Nowe cechy dla obiektu GATE:
    • Cecha OverloadDetection - możliwość wykrywania przeciążenia procesora za pomocą czerwonej diody.
    • Cecha ResetReason - funkcja diagnostyczna, określa przyczynę restartu modułu.
  • Dodanie czasu UTC do listy stref czasowych.
  • Poprawa stabilności i wydajności pracy oraz poprawki błędów.
