Co nowego

  • Dodano widok diagnostyczny dla urządzeń CLU
  • Dodano tryb ciemny dostępny pod Linuksem i MacOS
  • Dodano możliwość tworzenia kopii interfejsu myGrenton
  • Dodano akcję kopiuj / wklej dla widżetów myGrenton
  • Dodano widżet myGrenton ON_OFF_DOUBLE z dwoma przyciskami
  • Dodano również szablony widżetów, które dostępne są poprzez mechanizm Drag&Drop
  • Dodano szablony widżetów VALUE, ON_OFF, SCENE dla obiektów SatelZone, SatelInput, SatelOutput
  • Dodano szablon widżetu ON_OFF dla obiektów Timer
  • Dodano szablon widżetu VALUE dla obiektów ONEW_SENSOR

  • Zmieniono nazewnictwo obiektów z „obiekty peryferyjne” na „obiekty wirtualne”
