Historia Zmian

v1.0.1 (build 1932)

3 Wrzesień 2019

  • Nowy obiekt wirtualny Timer – obiekt, który może być wykorzystywany wszędzie tam, gdzie potrzebne jest wywołanie metody po określonym czasie lub jej cykliczne wywoływanie. Timer może pracować w dwóch trybach: Countdown oraz Interval.
  • Nowe cechy / metody obiektu GATE HTTP:
    • PrimaryDNS, SecondaryDNS – nowe cechy zwracające adres IP preferowanego oraz alternatywnego serwera DNS wykorzystywane przez moduł GATE.
    • SetPrimaryDNS, SetSecondaryDNS – nowa metoda ustawiająca adres IP preferowanego oraz alternatywnego serwera DNS.
    • Optymalizacja obsługi równoległych (kolejkowania) zapytań dla obiektu HttpListener.
    • Poprawa stabilności działania urządzenia.
