Grenton Object Manager

Recommended hardware requirements. Installation and start-up process

Recommended hardware requirements

Windows

  • Operating system: Windows 7 or higher
  • RAM Memory: at least 4 GB
  • Available disk space: at least 500 MB

macOS

  • Operating system: macOS 10.12 Sierra
  • RAM Memory: at least 4 GB
  • Available disk space: at least 500 MB

Linux

  • Operating system: Ubuntu 18.04 or higher
  • RAM Memory: at least 4 GB
  • Available disk space: at least 500 MB

Note!! To be able to use Object Manager v1.3.3, it is required to have at least 4.07.34 firmware on the CLU central unit.

Installation and start-up process

Windows

  • Download file .exe
  • Run downloaded file.
  • Select the Object Manager installation path.
  • Begin unpacking by clicking the Extract button.
  • After unpacking, run the om.exe file located in the /object-manager directory

macOS

The application name contains the version number, which allows multiple OM versions to exist on one computer. Before removing previous versions, it is recommended to migrate / copy project files stored in the application file by default. To do this, in the Finder, select (Ctrl-Click) "Show Package Contents" and copy or move the folder containing the project files (om.app/Contents/MacOS/projects) to the new version of the OM application. - Download the file. - Run the downloaded file.. - Copy the OM application to the Application folder as suggested. - Start the OM application in the standard way.

Linux

  • Download file .tar.gz.
  • Extract the downloaded file in the selected location.
  • After unpacking, run the ./om file located in the /object-manager directory.
