Installation and start-up process

  • Run the Object Manager program (version v1.3.2 or higher).
  • Open the last submitted project.
  • Make sure that no changes have been made to the project - there is no "*" next to the CLU icon.
  • Update the interface database:
    • Select "Tools" from the menu bar.
    • Select the „interface base” option.
    • Select the "Update interfaces base" option from the displayed list.
    • Make a configuration change, and then send it to the CLUZ module.
    • Perform a firmware update on CLUZ:
    • Select „Tools” from the menu bar.
    • Select “Update firmware”
    • Select the CLUZWAVE2 type object. Selecting the check box is only possible if the current firmware on the CLU is out of date.
    • Select „Update”
    • After reading the message, select "Yes" - after accepting, the update process will start.

Note!! During the upgrade process, do not turn off the power or perform other activities on the system. For the update to be successful, you must complete it to the end.

  • If the update is successful, [UPDATED] appears when the CLU is updated correctly.
  • Close the update process window.
  • If the update was successful, the firmware version after refreshing should be in accordance with selected to update.
  • Perform „CLU Discovery”.

Note!! After CLU Discovery is completed, you may receive a message regarding the configuration being sent to CLUZ. If the message occurs, click "OK", finish the Discovery process and then reload the interface database.

  • Reload interfaces base:
    • Select „Tools” from the menu bar.
    • Select „Update firmware”.
    • Select the option "Reload interface interface" from the displayed list.
    • Apply changes by selecting "Yes".
    • Make a configuration change, and then send it to the CLUZ module.
