Installation and start-up process

Note!
To use the CLU firmware version 04.07.49, it is required to have the Object Manager application version v.1.2.0 on the computer. If your application does not have the version mentioned, please contact our Technical Support Department.
  • Start the Object Manager program from which the configuration currently located in the central unit was sent.
  • Open the last project.
  • Open the CLU module window by double-clicking its icon in the module tree on the left side of the Object Manager software window.
  • Go to the "Embedded features" tab, check and save the FirmwareVersion value (at the bottom of the list) - it is necessary for later verification of the correctness of the procedure. The description of finding the value of the built-in feature can also be found at the following address: https://support.grenton.pl/pl/support/solutions/articles/35000042415-jakznukaj-informacje-o-wersji-oprogramowania-firmware-jednostki-centralnej-clu-z-wave
  • Make sure that no changes have been made to the project - there is no * symbol next to the CLU icon.
  • Perform interface database update:
    • Select "Tools" from the menu bar.
    • Select the "Interface database" item.
    • From the displayed list, select the "Update interface database" item.
  • Resend the configuration to the CLU.
Note!
If the configuration is up-to-date and the following message is displayed: Configuration on all CLUs is already up-to-date change in the project configuration. The fastest method is to change the name of a given input or output and confirm it with the OK button.
  • Perform firmware update on CLU:
    • Select "Tools" from the menu bar.
    • Select the option "Update firmware on CLU".
    • Highlight the CLU Z-Wave module.
    • Select "Update".

If the update was successful, the firmware version after refreshing should match the target version.
