Update process

Update process

Note!
The Smart Panel with firmware version 04.03.04 works only with CLU not older then CLU Z-Wave firmware version 04.07.49 (1912) and application Object Manager version 1.2.1-1902.
Note!
Before Smart Panel firmware update please update CLU Z-Wave firmware and interfaces database first. Update steps:
- The CLU Z-Wave firmware update.
- The XML interfaces database update.
- The Smart Panel firmware update (steps below).
- CLU Discovery retry.
Note!
To activate the new functionality of the Smart Panel steps below, before CLU Discovery, you shall:
- Remember/Copy current configuration of the Smart Panel,
- Remove all objects from the list of the Smart Panel,
- Remove permanently all objects of the Smart Panel form trash bin.
- Execute CLU Discovery.

Smart Panel firmware update via SD card:

  • Switch OFF the Smart Panel from the power supply.
  • Remove the SD card from the Smart Panel slot according to the instruction documentation.
  • Save the new firmware files in the SD card directory „upload0303”.
  • Insert the SD card to the Smart Panel slot back.
  • Switch ON power supply of the Smart Panel.
  • After a few seconds of bootup the Smart Panel finds the new firmware and starts update process.
  • Update process is signaled by fast alternating blinking of the red and green LEDs.
  • Update success is signaled by few blinks of the green LEDs.
  • In case of failure please try update process again.

In case of constant failures while firmware update please contact with the Grenton Technical Support.
