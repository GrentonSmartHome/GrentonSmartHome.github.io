WiFi Smart Panel Firmware

Installation and start-up process

Update process

Note!
For Smart Panel WiFi with firmware version 5.9.1-2033, it is necessary to have CLU Smart Panel WiFi firmware version 1.1.5-2033 (or higher).
Note!
Before Smart Panel WiFi firmware update please update CLU Smart Panel WiFi firmware and interfaces database first. Update steps:
- CLU Smart Panel WiFi firmware update.
- Updating and reloading the XML interfaces database.
- Updating the Smart Panel WiFi firmware (according to the Smart Panel WiFi update procedure below).
- Execution of CLU Discovery.

Smart Panel WiFi firmware update via SD card:

  • Disconnect the Smart Panel WiFi from the power supply.
  • Download the .zip file and extract it.
  • Remove the microSD card from the Smart Panel WiFi memory slot according to the procedure presented in the manual or in the data sheet.
  • Paste the downloaded (unpacked) files onto the memory card, overwriting the existing ones.
  • Put the memory card back into the slot.
  • Connect the power supply to the Smart Panel WiFi module.
  • After powering and detecting a newer firmware version, the module will start the update procedure.
  • The software update process is signaled by alternating flashing of the upper diodes.
  • Correct completion of the update is confirmed by several blinks of all LEDs green.
  • In case of failure of the update process, the whole procedure should be repeated.

In case of constant failures while firmware update please contact with the Grenton Technical Support.
