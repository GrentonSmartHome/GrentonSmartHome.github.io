Beta Version

7.0.6 (build 2345)

8 January 2025

Firmware is accessible from the Smart Panel v7 package.

Update process.

Functionality description: Smart Panel v7 - User Manual.

What's New

  • Improved font appearance.
  • Added CoolMaster support on the thermostat page. Control including temperature adjustment, supported operating modes, and on/off.
  • Added heating/cooling icon on the thermostat page.
  • Changed temperature adjustment step in the thermostat from 0.1 to 0.5.
  • Added the StartCleaningMode method, which locks Smart Panel buttons for a specified time (Panel cleaning function).
  • Optimized Smart Panel performance (sped up thermostat page loading - Smart Panel now queries the thermostat feature group instead of individual features).
Note!
Smart Panel FW version 7.0.6-2345 is supported be the CLU Z-Wave 2.0 FW version 5.14.01 (or higher).
