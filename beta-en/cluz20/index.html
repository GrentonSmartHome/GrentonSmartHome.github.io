CLUZ 2.0 Firmware

Download the beta software and find out what's new

Beta Version

v5.09.01 (build 2150)

04 January 2022

Firmware is accessible from the zip package.

Installation and start-up process in accordance with the User Manual - Chapter VI, Section 10.2.B

What's New

  • Support for Multisensor module:
    • IR transceiver/receiver,
    • Temperature sensor,
    • Air humidity sensor,
    • Atmospheric pressure sensor,
    • Light sensor,
    • Sound sensor,
    • Air CO2 sensor,
    • Air VOC sensor.
  • Support for Roller Shutter x1 in 3.1.2 version:
    • Reversing of Roller Shutter direction,
    • Reversing of range of shutter position (0-100% for 100-0%),
    • Compensation of shutter actuation time,
    • Features that determine the upward and downward movement time of the roller shutter,
    • State of Roller Shutter calibration,
    • Option to specify target lamel position for SetPosition method - Default (lamel position in line with the roller movement), Previous (previous lamel position) or chosen by user,
    • Option for Calibration direction,
    • New events OnPositionChanged and OnLamelPositionChange.
  • New virtual object - Event Scheduler:
    • Ability to set events in the system that will be triggered based on the applied CRON.
  • Sunrise Sunset Calendar:
    • Features informing about the next sunrise/sunset,
    • Feature informing about whether it is currently day or night,
    • Features providing sunrise / sunset times as unix and local time,
    • Ability to add an offset to the event before and after sunrise and sunset.
  • Presence Sensor:
    • 2 modes of presence sensor: impulse and state mode,
    • This function allows the operation of a Presence Sensor object with a switch.
  • New event OnTimeChange in CLU.
Note!
To be able to update CLU to v5.09.01, you must have Object Manager v1.6.0 (or higher)
