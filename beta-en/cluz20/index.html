What's New

v5.15.12 (build 2607)

12 February 2026

  • Improvement in the handling of DAPCValue in DALI_GEAR objects.

v5.15.10 (build 2551)

13 January 2026

  • LED RGBW DIN version 3.0.5 and LED RGBW FM version 2.0.1:
    • New LED_CHANNEL objects enabling control of individual module channels.
    • DistributedLogic mode for single channels.
    • Improvement in the area of ​​device communication on the TF-Bus (problem with a disappearing module).
Note!
The RedValue, GreenValue, BlueValue and WhiteValue features and the methods associated with these features (Set / Hold) have been hidden in the main LEDRGBW object.

v5.15.09 (build 2551)

18 December 2025

  • DALI Controller version 3.0.6:
    • Added new method GetDAPCValue in DALI_MASTER and DALI_GEAR objects.
    • Changed the mechanism for querying ballasts for the current DAPC value, currently performed during module restart, calling the GetDAPCValue() method, and at the end of calling the Hold method.
    • Changed the way the DAPCValue feature is set, currently set after sending the command to CLU and after receiving a report from the controller.
    • Optimized the number of DALI Controller module reports transmitted over TF-Bus.
