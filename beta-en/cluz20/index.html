CLUZ 2.0 Firmware

Download the beta software and find out what's new

Beta Version

v5.08.01 (build 2128B)

28 July 2021

Firmware is accessible from the zip package.

Installation and start-up process in accordance with the User Manual - Chapter VI, Section 10.2.B

What's New

  • Support for Smart Panel V6.
  • Support for Roller Shutter x3 in 2.1.3 version:
    • Percentage value of the shutter opening (0 - 100%).
    • Roller shutter lamel position (0° - 90°).
    • Method for roller shutter calibration.
    • Option to Enable / disable the ability to control the roller shutter.
    • OnLamelOpen and OnLamelClosed event.
  • Support for Dimmer DIN in 1.1.11 version:
    • Supply frequency 50/60Hz autodetection.
    • Dimming functionality improvements.
  • Stability improvements for modules:
    • Analog IN/OUT DIN in 1.2.13 version.
    • Digital IN DIN in 1.2.14 version.
    • I/O Module DIN in 1.4.11 version.
    • LED RBGW DIN in 1.4.7 version.
    • Relay 2HP DIN / Relay 4HP DIN in 1.3.13 version.
    • I/O Module FM in 1.1.9 version.
    • Roller Shutter FM in 2.1.3 version.
  • Support for Z-Wave Aeotec Nano Shutter v3.04 module.
Note!
To be able to update CLU to v5.08.01, you must have Object Manager v1.4.1 (or higher)
