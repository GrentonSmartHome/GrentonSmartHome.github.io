What's new

  • Added myGrenton widget TV_REMOTE_CONTROL.
  • Added myGrenton widget EVENT_SCHEDULER.
  • Added myGrenton widget AUDIO_REMOTE_CONTROL.
  • Added myGrenton widget MULTISENSOR and widget template for MULTISENSOR_IR_FM objects.
  • Added myGrenton widget ROLLER_SHUTTER_V3.
  • Added myGrenton widgets VALUE_V2 and VALUE_DOUBLE.
  • Added new icons in myGrenton: multisensor, remote, scheduler, text, value-double, value.
  • Added clearing of clipboard with widgets content when opening new project.
  • Added dialog to display the result of method execution (e.g. for 'EventScheduler' object).
  • Added icon visibility change for TEXT widget depending on configuration.
  • Added context action to create copy of script.
  • Added find/replace functionality in LUA script editor.
  • Added translation of application into Italian.
  • Added demo projects.
  • Changed the way of defining indications for the ON_OFF DOUBLE widget - with the use of separate attributes: Indication ON, Indication OFF.
  • Changed default units for component of empty LED widget.
  • Changed order of attributes in widgets SCHEDULER and TEXT.
  • Changed wizard for CLU Discovery, i.a. added object updates after firmwareApiVersion changed.
  • Fixed widget selection behavior when scrolling the page in myGrenton.
  • Fixed default width of column in properties for myGrenton widget.
  • Fixed shortcuts in LUA script editor.
  • Fixed refreshing of values in embedded features tabs.
  • Fixed necessity to enter private key after rewriting active CLU to inactive.
  • Fixed refreshing of embedded features tab after changing values from combo, refresh only occurs for the feature StatisticState.
  • Fixed presentation of signal value for Z-Wave devices in diagnostic view (from -1 to N/A).
  • Fixed THERMOSTAT widget in the context of Min / Max range.
  • Fixed renaming with F2 shortcut - problems with updating the object.
