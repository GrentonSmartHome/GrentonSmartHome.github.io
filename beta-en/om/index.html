What's new

v1.5.1 (build 214101)

  • Added warning about ending Visual Builder support.
  • Improved display of large user features.
  • Fixed display of the Connection attribute for TF-BUS devices in diagnostic view.
  • Fixed saving problems with the Home Manager interface.
  • Removed improvement of passing the LUA variable value to the remote CLU (functionality will be available in the next version 1.6.0 after additional testing).
  • Removed improvement in generating LUA code from OML containing comments (functionality will be available in the next version 1.6.0 after additional testing).

v1.5.0 (build 212301)

  • Added editing of object names using shortcut F2.
  • Added filtering of device objects in the tree by name.
  • Added SCHEDULER widget.
  • Added ROLLER_SHUTTER, ROLLER_SHUTTER_V2 widget templates for ROLLER_SH_DIN_3, ROLLER_SH_FM device objects.
  • Added TEXT text.
  • Fixed functionality of passing the variable value to the remote CLU in LUA scripts.
  • Fixed functionality of generating LUA code from OML containing comments.
  • Fixed translations.
