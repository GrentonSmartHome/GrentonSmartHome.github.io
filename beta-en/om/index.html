What's new

v1.12.1 (build 2516)

14 May 2025

  • Improved presentation of the number of CLU objects:
    • Removed warning information about exceeding the number of objects.
    • Added information about the number of module objects and virtual objects in the tooltip.
  • Removed Configuration Schemes tab in the Object Properties view.
  • Improved stability and performance of work in the project tree during editing.

v1.12.0 (build 2509)

17 March 2025

  • Added new widget INTERCOM, which allows you to make a call from the myGrenton application to the SIP intercom.
Note!
Using the INTERCOM widget requires the myGrenton application version 1.16.0 (or higher) for iOS app or 1.14.14 (or higher) for Android app.
