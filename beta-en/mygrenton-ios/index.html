What's New

1.17.0 (251503)

15 May 2025

  • Improved statistics functionality:
    • Added support for statistics from the Statistics and StatisticsPulse virtual objects.
    • Fixed minor UI bugs.
  • Bug fixes:
    • Unstable app behavior when trying to display the view responsible for thermostat control.
    • App crash when attempting to add an interface via QR code.
    • App crash in intercom settings.
    • Fixes in interface settings.

1.16.0 (250902)

17 March 2025

  • Added an intercom widget to perform outgoing calls from myGrenton.
  • Added the ability to call back missed intercom calls from the call history and push notifications.
  • Improved app stability.
  • Updated license content.
  • Minor UI fixes.

1.15.0 (245102)

8 January 2025

  • Added support for statistics for media measurements and sensors.
  • Fixed crash when quickly switching sliders in the settings view.
  • Added saving of slider settings (UseCloud, Push, Statistics) after overwriting the interface.
  • Fixed bug related to handling remote camera stream.
  • Fixed bug where the side menu didn't close after changing pages.
  • Changed the color of the Remove Interface button to red.
