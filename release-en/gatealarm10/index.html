Changelog

v1.1.0 (build 2012)

28 September 2020

  • Integration of the Jablotron alarm control - enables integration of the Grenton system with the Jablotron alarm control (JA-101). Integration of the Grenton system with the Jablotron is possible via the JA-121T module. New virtual objects are used for configuration:
    • Jablotron - allows configuration to be made to integrate the system with the Jablotron alarm control;
    • JablotronSection - allows creating a section to which access will be possible after entering the password of one of the users or the password of the administrator himself;
    • JablotronDevice - gives the ability to monitor the status of the selected input / device;
    • JablotronOutput - allows monitoring and setting the status of the selected output after entering the user or administrator password.
    • New virtual object Timer - an object that can be used wherever you need to call a method after a specified time or its cyclical calling. The timer can work in two modes: Countdown and Interval.
    • Improved handling of incorrect incoming frames from the Satel alarm control.
    • Improved device stability.
