Changelog

v1.2.3 (build 2310)

10 July 2023

  • Added support for special characters in WiFi password and SSID.
  • Unified behaviour of the ROLLER SHUTTER Hold() method.
  • Fixed ROLLER_SHUTTER object getting stuck (State=3) when MaxTime > 25s.
  • ROLLER_SHUTTER object events order changed to correct.
  • DIN object events order changed to correct.
  • Bug fixes, improved device stability and performance.
NOTE!
To be able to update WiFi modiule to v1.2.3, you must have Object Manager v1.6.1 (or higher).

v1.1.5 (build 2137D)

17 August 2022

  • Added support for an AC current frequency of 60Hz at a phase voltage of 110..230V.
