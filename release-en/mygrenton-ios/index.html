Changelog

1.17.0

23 June 2025

  • Added support for statistics.
  • Added an intercom widget to perform outgoing calls from myGrenton.
  • Added the ability to call back missed intercom calls from call history and push notifications.
  • Updated license.
  • Improved app stability.
  • Fixed an issue where settings options were reset after overwriting the interface.
  • Introduced UI/UX improvements.

1.14.1

3 October 2024

  • Refreshed thermostat control view.
  • Added support for displaying modes: holiday, heating - to the thermostat widget.
  • Added new thermostat widget (v2) with additional control options:
    • Heating/cooling.
    • Fan mode.
  • Improved appearance and navigation of the views used to manage widgets:
    • Refreshed TV and audio remote control views.
    • Added swipe down gesture - for easier closing of views.
    • Views do not stretch to full screen width on iPads.
  • Improved presentation of menus with interface pages on iPads:
    • Added button in top navigation bar.
    • Added swipe to right gesture at the left edge of the screen.
  • Improved parsing of values received from CLU for widgets with scheduler.
  • Improved app stability.
  • Fixed an error related to adding an interface for devices with iOS15.
  • Minor UI/UX improvements.
