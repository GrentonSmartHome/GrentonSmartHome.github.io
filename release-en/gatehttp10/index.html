Changelog

v1.0.1 (build 1932)

3 September 2019

  • A new virtual object Timer - an object that can be used wherever it is necessary to call a method after a specified time or its cyclic execution. The timer can work in two modes: Countdown and Interval.
  • New features / methods of the GATE HTTP object:
    • PrimaryDNS, SecondaryDNS - new features that return the IP address of the preferred and alternative DNS server used by the GATE module.
    • SetPrimaryDNS, SetSecondaryDNS - new method to set the IP address of the preferred and alternate DNS server.
    • Optimization of handling of parallel (queuing) queries for the HttpListener object.
    • Improving the stability of the device operation.
