What's New

v0.7.0 (build 2604)

26 January 2026

Fixes:

  • Fixed an issue where the blockMotion method of the RollerShutter object was called incorrectly.
  • Fixed an issue causing long startup times for the CLU on certain module configurations.

v0.6.0 (build 2602)

19 January 2026

New features:

  • Added the emitEvent function to generate events from the controller.

Fixes:

  • Fixed an issue with incorrect CLU status after a failed update.
  • Fixed an issue with outlet selection when using object functional types.
  • Fixed incorrect behavior of the _first function.
  • Fixed incorrect logical functions within outlet selection.

v0.4.0 (build 2551)

5 January 2026

Fixes:

  • Dimmer DIN module: Fixed the device discovery process.
  • TP4 / TP8 / SP modules: Fixed module behavior at runtime.
  • led object: Fixed an issue with physical output control at runtime.
  • led object: Fixed a freeze issue when rapidly calling the changeBrightnessPercent / changeLedBrightness methods across the full range.

v0.3.0 (build 2550)

15 December 2025

Fixes:

  • dimmer object: Fixed generation of the onIsOnChange event.
  • dimmer object: Fixed generation of the onDimmerBrightnessRise / onDimmerBrightnessLower events when calling methods other than switch/on/off.
  • dimmer object: Fixed setting the brightnessPercent value after calling the changeBrightnessPercent / changeDimmerBrightness methods.
  • led object: Fixed reporting of the distributedLogicGroup attribute.
  • led object [Led FM]: Fixed value changes when calling the switch/on/off methods.
  • led object [Led FM]: Fixed returning the brightnessPercent / ledBrightness values after calling the changeBrightnessPercent / changeLedBrightness methods.
  • led object [Led FM]: Fixed operation of the increaseBrightnessPercent and decreaseBrightnessPercent methods.
  • roller-shutter object: Fixed operation of the blockMotion and startCalibration methods.
  • dout object: Fixed returned value of the powerValue feature.
  • switch-for-duration protocol: Fixed switching on/off for a specified time when the provided parameter is of type int.

Known issues:

  • Connection issue with CLU if the Touch Panel 4B / Touch Panel 8B was added using New Project â€” a system restart is required for the connection to work properly.
