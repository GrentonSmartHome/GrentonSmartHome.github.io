What's New

v0.5.8

12 January 2026

New features:

  • Added the ability to edit myGrenton labels from the project settings.

Fixes:

  • Fixed an issue where the update notification was shown again after being previously dismissed.

Improvements:

  • Added the ability to edit tag labels directly from the table.
  • Improved the update page accessible from the project settings.
  • Redesigned the tag creation wizard for new projects.

15 December 2025

  • Fixed the display of the application update notification.

Improvements:

  • Improved the styling of UI components.

v0.5.6

8 December 2025

Fixes:
