What's New

v0.6.3

23 February 2026

Fixes:

  • Fixed incorrect object counting in the outlet for the Select Objects option.
  • Fixed an issue where the controller icon and initial property values were reset after reopening the project.
  • Fixed an issue where focus was lost after clearing content in multiple search fields.
  • Fixed incorrect search behavior in the tag selection window.

Improvements:

  • Added millisecond display in diagnostic logs.
  • Added category label display in the tag selection window.
  • Changed tag display in the action tree on the right side of the editor to a hierarchical view.
  • Changed tag display in the outlet object selection window to a hierarchical view.

v0.6.2

16 February 2026

Fixes:

  • Fixed missing quick actions in the Overview window for the Analog In/Out module.
  • Fixed missing translations in some dialog windows.
  • Fixed an issue with the context menu being displayed in the corner of the screen in the My Projects table.
  • Fixed missing validation for tag names starting with a digit.
  • Fixed page scrolling issue in the user library tab under Profile.
  • Fixed inability to log in to the system via the status button in the top bar when browser tokens had expired.

Improvements:

  • Improved the jumping behavior of the step navigation window when creating a new project.
  • Changed the tag display in the tag settings table to a hierarchical view.
  • Added a dropdown list for methods and features in Runtime that have predefined values in the protocol.

v0.6.1

11 February 2026

New features:

  • Added a virtual object Scheduler.
  • Added controller selection and automatic assignment of Grenton standard controllers to rooms when creating a new project.

Fixes:

  • Fixed an issue preventing changes to user roles.
  • Fixed missing label incrementation when creating multiple rooms in the location tag wizard.
  • Fixed incorrect object count display in the outlet tree for automatic outlet selection in the controller when no objects were present.
  • Fixed incorrect behavior of the Light Group controller.
  • Fixed an issue where a controller could select itself in outlet selection/auto-selection.
  • Fixed an issue where selected outlet options were not being saved when editing an outlet.

Improvements:

  • Moved the Runtime view to object preferences — live object control is now available directly within the object’s tab.
  • Changed project tag display to a hierarchical label-based view.
  • Updated the new project creation flow — after completing Discovery, users are now redirected to the Preview window.

  • Improved the user authorization dialog.
