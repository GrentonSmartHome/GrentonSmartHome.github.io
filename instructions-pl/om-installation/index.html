Grenton Object Manager

Zalecane wymagania sprzętowe, proces instalacji i uruchomienie

Zalecane wymagania sprzętowe

Windows

  • System operacyjny: Windows 7 (64-bit) lub wyższy
  • Pamięć RAM: min. 4 GB
  • Dostępna przestrzeń na dysku: min. 1 GB

macOS

  • System operacyjny: macOS 10.12 Sierra
  • Pamięć RAM: min. 4 GB
  • Dostępna przestrzeń na dysku: min. 1 GB

Linux

  • System operacyjny: Ubuntu (64-bit) 18.04
  • Pamięć RAM: min. 4 GB
  • Dostępna przestrzeń na dysku: min. 1 GB
Uwaga!
Aby możliwe było korzystanie z Object Managera v.1.3.0 (lub wyższej), wymagane jest posiadanie na jednostce centralnej CLU firmware w wersji co najmniej 4.07.34.

Proces instalacji i uruchomienie

Windows

  • Pobrać plik .exe
  • Uruchomić pobrany plik.
  • Wybrać ścieżkę instalacji Object Managera.
  • Rozpocząć wypakowywanie klikając w przycisk Extract.
  • Po wypakowaniu uruchomić plik om.exe znajdujący się w katalogu /object-manager.

macOS

Nazwa aplikacji zawiera numer wersji co umożliwia istnienie wielu wersji OM na jednym komputerze. Przed usunięciem poprzednich wersji zalecane jest wykonanie migracji/kopii plików projektów domyślnie przechowywanych w pliku aplikacji. W tym celu należy w programie Finder wybrać opcję (Ctrl-Click) “Pokaż Zawartość Pakietu” (Show Package Contents) i skopiować lub przenieść folder zawierający pliki projektu (om.app/Contents/MacOS/projects) do nowej wersji aplikacji OM.

  • Pobrać plik
  • Uruchomić pobrany plik
  • Skopiować aplikację OM do folderu Aplikacji zgodnie sugestią
  • Uruchomić aplikację OM w standardowy sposób

Linux

  • Pobrać plik .tar.gz
  • Wypakować pobrany plik w wybranej lokalizacji
  • Po wypakowaniu uruchomić plik ./om znajdujący się w katalogu /object-manager
